# C++ Simple Text Editor

A minimal console-based text editor built in **C++** during my internship.

## Features
- Create new text files
- Open existing files
- Edit content line by line
- Save changes
- Display file with line numbers

## How to Run
```bash
g++ editor.cpp -o editor
./editor
